# nanobot-skill-weather

Weather information skill for [Nanobot AI](https://github.com/kombalarasoftware-cmd/MyBot).

## Installation

Install via the Nanobot AI Dashboard:

1. Go to **Settings → Skills → Marketplace**
2. Search for "weather"
3. Click **Install**

## Features

- Current weather conditions for any city
- Multi-day weather forecasts (up to 7 days)
- Severe weather alerts and warnings
